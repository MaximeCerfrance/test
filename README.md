# Time Capsule SaaS

Prototype d'application web de capsule temporelle numérique. Cette base utilise Next.js pour le front-end et une API Node.js. Le but est de fournir un point de départ pour une plateforme sécurisée avec déverrouillage différé.

## Scripts

- `npm run dev` : démarrer le serveur de développement.
- `npm test` : exécuter les tests (aucun test pour l'instant).

## Fonctionnalités prévues

- Création de capsules avec données chiffrées.
- Déclenchements différés (date, décès, événements).
- Intégration blockchain pour horodatage.
- Interface claymorphism.

Ce dépôt contient uniquement une maquette minimale.
